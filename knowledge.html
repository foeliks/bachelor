Einführung

JavaScript ist...

  • Dynamisch typisiert
    Variablen können unterschiedliche Datentypen aufnehmen
  • Objektorientiert
    Es gibt Objekte, Vererbung, Polymorphie
  • Prototypen-basiert
    Objekte werden über Prototypen konstriert

JavaScript kann in HTML über drei Möglichkeiten eingebunden werden:

  1. In externen Dateien im Header
    <script src="Dateiname.js" />
  2. Einbettung im Dokument an beliebiger Stelle
    <script type="text/javascript"> JavaScript Code </script>

    Code wird sofort ausgeführt, wenn der Webbrowser die Stelle einliest. Es kann also sein, dass Elemente, die "weiter unten" im Dokument noch nicht ansprechbar sind
    deshalb: Am Ende des <body> Tags einfügen
  3. In speziellen Attributen in HTML-Elementen als Aktionen
    <button onclick=" JavaScript Code ">Cooler Buttonname</button>

Kommentare können wie in anderen Programmiersprachen mittels ...

  • // Einzeiliger Kommentar
  • /*
    Mehrzeiliger
    Kommentar
    */

... eingefügt werden

Eine Ausgabe von Inhalten kann auf zwei Weisen erfolgen:

  1. alert: Gibt eine PopUp-Meldung in Form eines Dialogfensters im Webbrowser aus
  2. console: Gibt Nachrichten in der Entwicklerkonsole des Webbrowsers aus
    • .log()
      Nachricht wird ohne weiteres Highlighting ausgegeben
    • .warn()
      Nachricht wird als Warnung (häufig gelb) ausgegeben
    • .error()
      Nachricht wird als Fehler (häufig rot) ausgegeben

Es gibt einen "Strict Mode", der ...

  • Fehler meldet, sobald z.B. auf nicht deklarierte Variablen zugegriffen wird
  • Dich vor Fehlern bewahrt, die schwer zu finden sein können
  • Über "use strict;" in der ersten Zeile des Codes aktiviert wird
    (Du musst diesen nicht selbst aktivieren, das hat der nette Entwickler dieser Lernumgebung schon für dich gemacht)

Datentypen

JavaScript kennt die folgenden primitiven Darentypen:

  • string (Zeichenketten)
  • number (Zahlenwerte)
  • boolean (wahr | falsch)

Spezielle Werte:

  • null (Kein Wert)
  • undefined (Nicht definierter Wert)

Strings

JavaScript bietet ab ES6 viele Funktionen nativ an, um komfortabel mit Strings arbeiten zu können

var str = 'This is a "message';
var str = "This is a 'message";

 Sonderzeichen wie " (doppelte Anführungsstriche) und ' (einfache Anführungsstriche) müssen je nach Verwendung maskiert werden

' => \'
"=> \"
\ => \\

z.B.: var str = 'This is a \'message';

 Methoden/Attribute

 Methode/AttributBeschreibung.
 .lengthGibt die Länge des String zurück
.indexOf(needle)Gibt die erste Position des Strings needle zurück. -1, fallsnichts gefunden wurde.
.lastIndexOf(needle)Gibt die erste Position des letztenn Vorkommens des Strings needle zurück. -1, falls nicht gefunden wurde.
  .substring(startpos, length) Gibt einen neuen String mit dem Teilstring von startpos mitder Länge length zurück
 .replace(str, newstr)Gibt einen neuen String zurück, in dem die Zeichenkette in str durch die in newstr ersetzt ist.

 Weitere: trim(), concat(), charAt(), toLowerCase(), toUpperCase(), ...

Kombination von Zahlen und Zeichenketten

Wird eine Zahl zu einem String (oder ein String zu einer Zahl) addiert, so wird die Zahl zu einemString umgewandelt

  • + (Plus) ist auch das Zeichen für Konkatenation in JavaScript
  • Die Ausführung geschieht von links nach rechts
  • Sobald ein String beteiligt ist, erfolgt eine Konkatenation

(Konkatenation bedeutet, dass zwei Listen bzw. Strings aneinander gehängt werden)

Kontrollstrukturen

JavaScript bietet die folgenden Kontrollstrukturen an:

  • If-Else (bedingte Ausführung)
  • Switch-Case
  • While (Schleife)
  • For (Schleife)
  • Do ... While (Schleife)

If-Else wird wie folgt definiert:

if (/* Bedingung */) {

// Anweisung 1

} else if (/* Bedingung */) {

// Anweisung 2

} else {

// Anweisung 3

}

Switch-Case wird wie folgt definiert:

switch (/* Variable */) {

case /* Wert 1 */:

// Anweisung 1
break;

default:

// Anweisung
break;

}

While (Schleife) wird wie folgt definiert:

while (/* Bedingung */) {

// Anweisung

}

For (Schleife) wird wie folgt definiert:

for (/* Initialisierung; Bedingung; Wertänderung */) {

// Anweisung

}

Do ... While (Schleife) wird wie folgt definiert:

do {

// Anweisung

} while (/* Bedingung */);

In JavaScript ist es möglich, mit break die Schleifendurchführung zu unterbrechen oder mit continue wieder an den Schleifenanfang zu springen (bzw. den aktuellen Durchlauf zu überspringen).

Beispiel:

let i = 10;

while (i > 0) {

if (i == 3) {

break;

}

}

alert(i); // Ergibt 3
