Einführung

JavaScript ist...

  • Dynamisch typisiert
    Variablen können unterschiedliche Datentypen aufnehmen
  • Objektorientiert
    Es gibt Objekte, Vererbung, Polymorphie
  • Prototypen-basiert
    Objekte werden über Prototypen konstriert

JavaScript kann in HTML über drei Möglichkeiten eingebunden werden:

  1. In externen Dateien im Header
    <script src="Dateiname.js" />
  2. Einbettung im Dokument an beliebiger Stelle
    <script type="text/javascript"> JavaScript Code </script>

    Code wird sofort ausgeführt, wenn der Webbrowser die Stelle einliest. Es kann also sein, dass Elemente, die "weiter unten" im Dokument noch nicht ansprechbar sind
    deshalb: Am Ende des <body> Tags einfügen
  3. In speziellen Attributen in HTML-Elementen als Aktionen
    <button onclick=" JavaScript Code ">Cooler Buttonname</button>

Kommentare können wie in anderen Programmiersprachen mittels ...

  • // Einzeiliger Kommentar
  • /*
    Mehrzeiliger
    Kommentar
    */

... eingefügt werden

Eine Ausgabe von Inhalten kann auf zwei Weisen erfolgen:

  1. alert: Gibt eine PopUp-Meldung in Form eines Dialogfensters im Webbrowser aus
  2. console: Gibt Nachrichten in der Entwicklerkonsole des Webbrowsers aus
    • .log()
      Nachricht wird ohne weiteres Highlighting ausgegeben
    • .warn()
      Nachricht wird als Warnung (häufig gelb) ausgegeben
    • .error()
      Nachricht wird als Fehler (häufig rot) ausgegeben

Es gibt einen "Strict Mode", der ...

  • Fehler meldet, sobald z.B. auf nicht deklarierte Variablen zugegriffen wird
  • Dich vor Fehlern bewahrt, die schwer zu finden sein können
  • Über "use strict;" in der ersten Zeile des Codes aktiviert wird
    (Du musst diesen nicht selbst aktivieren, das hat der nette Entwickler dieser Lernumgebung schon für dich gemacht)

Strings

JavaScript bietet ab ES6 viele Funktionen nativ an, um komfortabel mit Strings arbeiten zu können

var str = 'This is a "message';
var str = "This is a 'message";

 Sonderzeichen wie " (doppelte Anführungsstriche) und ' (einfache Anführungsstriche) müssen je nach Verwendung maskiert werden

' => \'
"=> \"
\ => \\

z.B.: var str = 'This is a \'message';

 Weitere: trim(), concat(), charAt(), toLowerCase(), toUpperCase(), ...

Kombination von Zahlen und Zeichenketten

Wird eine Zahl zu einem String (oder ein String zu einer Zahl) addiert, so wird die Zahl zu einemString umgewandelt

  • + (Plus) ist auch das Zeichen für Konkatenation in JavaScript
  • Die Ausführung geschieht von links nach rechts
  • Sobald ein String beteiligt ist, erfolgt eine Konkatenation

(Konkatenation bedeutet, dass zwei Listen bzw. Strings aneinander gehängt werden)

Variablen

Es gibt drei Wege im "Strict Mode" Variablen zu deklarieren:

  1. let
    Deklariert eine Variable im aktuellen Block (scope)
  2. const
    Deklariert eine konstante im aktuellen Block
  3. var
    Deklariert eine Variable im globalen Bereich

Ohne "Strict Mode" können Variablen beliebig deklariert und verwendet werden. Diese sind dann automatisch global.

Grundsätzlich: Variablen so lokal wie möglich halten

Variablennamen können...

  • a..z und A..Z (Buchstaben)
  • 0..9 (Zahlen)
  • _ (Unterstrich)
  • $ (Dollar-Zeichen)

enthalten, aber dürfen in ES6 nicht mehr mit einer Zahl beginnen

Namen sind case-sensitive (Groß- und Kleinschreibung beachten)

Reservierte Schlüsselworte können nicht als Namen verwendet werden (Wie z.B. if, for, ...)

Variablen ohne einen expliziten Wert, besitzen den Wert undefined

Die Überprüfung, ob eine Variable undefined ist, geschieht über den Operator typeof

if (typeof name === 'undefined') {

// Anweisung

}

Hinweis: Das hier verwendete If Konstrukt wird im Kapitel "Kontrollstrukturen" näher erklärt.

Eine Neu-Deklaration einer Variable überschreibt den alten Wert

let name = 1;

let name;

// name ist undefined

In JavaScript können Variablen nach ihrer Verwendung deklariert werden (Hoisting genannt):

x = 42;
console.log(x);
var x;

Hintergrund: JavaScript sortiert den Block vor der Ausführung so um, dass Deklarationen am Anfang stehen.

JavaScript sortiert nur die Deklaration (var x), nicht die Inititalisierung (= 42).

Kontrollstrukturen

JavaScript bietet die folgenden Kontrollstrukturen an:

  • If-Else (bedingte Ausführung)
  • Switch-Case
  • While (Schleife)
  • For (Schleife)
  • Do ... While (Schleife)

If-Else wird wie folgt definiert:

if (/* Bedingung */) {

// Anweisung 1

} else if (/* Bedingung */) {

// Anweisung 2

} else {

// Anweisung 3

}

Switch-Case wird wie folgt definiert:

switch (/* Variable */) {

case /* Wert 1 */:

// Anweisung 1
break;

default:

// Anweisung
break;

}

While (Schleife) wird wie folgt definiert:

while (/* Bedingung */) {

// Anweisung

}

For (Schleife) wird wie folgt definiert:

for (/* Initialisierung; Bedingung; Wertänderung */) {

// Anweisung

}

Do ... While (Schleife) wird wie folgt definiert:

do {

// Anweisung

} while (/* Bedingung */);

In JavaScript ist es möglich, mit break die Schleifendurchführung zu unterbrechen oder mit continue wieder an den Schleifenanfang zu springen (bzw. den aktuellen Durchlauf zu überspringen).

Beispiel:

let i = 10;

while (i > 0) {

if (i == 3) {

break;

}

}

alert(i); // Ergibt 3

Funktionen

JavaScript unterstützt Funktionen, diese müssen einen Wert über return zuückgeben

function myFunction() {

return 1;

}

Funktionen können Variablen zugewiesen werden

function myFunction() {

return 1;

}

let z = myFunction();

Funktionen können Parameter übergeben bekommen

  • Parameter werden über innerhalb der "(...)" definiert

funciton myFunction2(parameter) {

return parameter;

}

Funktionen können Standardwerte für Parameter festlegen

  • diese werden über den "=" Operator definiert

function myFunction(parameter = 3) {

return parameter;

}

Funktionen können Funktionen als Parameter verwenden 

  • die "Parameter Funkion" kann dann innerhalb der Funktion verwendet werden

function myFunction(fn, x, y) {

return fn(x, y);

}

let z = myFunction(

function(a, b) {

return a + b;

}, 

4, 5);

// z = 9

Arrays

Ein Array kann man sich wie eine Variable als Schachtel vorstellen. Allerdings hat diese Schachtel - im Gegensatz zu der einer einfachen Variablen - durchnummerierte Unterteilungen, in denen die Werte der einzelnen Elemente gespeichert werden.

Beispiel:

var arr = ['value1', 2, 3.5]; // alternativ: new Array(...)

Bei Arrays kann man ...

  • auf Elemente mittels Index (beginnend bei 0) zugreifen

var arr[1]; // Ergibt 2

  • Elemente hinten anfügen

arr.push(4); // Der Wert 4 wird hinten angefügt

  • Elemente von hinten entfernen und zurückgeben

var b = arr.pop(); // b = 4

Über ein Array kann mittels...

  • for-Schleife

for(let i = 0; i < arr.length; i++) { 

console.log(arr[i]);

}

  • forEach-Schleife

a.forEach(function(value, index) {

console.log(value); // alternativ console.log(arr[index]);

}

  • for...of

for(const element of arr) {

console.log(element); 

}

iteriert werden.

Hinweis: Alle Beispiele haben die gleiche Ausgabe in der Console
